

# HR Analytics Dashboard – Power BI

End-to-end HR analytics project built in Power BI, focused on **performance reviews, employee satisfaction and attrition analysis**.
The project demonstrates my ability to **design a star schema model, prepare data, write DAX measures and deliver decision-ready dashboards**.

---

## Data & Modeling

* Designed a **star schema** with a performance review fact table and multiple HR dimensions
* Built a **custom Date dimension** and managed **active / inactive relationships** to support multiple date contexts
* Reused rating and satisfaction dimensions across multiple measures to keep the model clean and scalable
* Optimized relationships and model structure for clarity and performance

(Model diagram included in the repository)

---

## Data Preparation

* Combined multiple source files into a single analytical model
* Performed data cleansing, standardization and enrichment in **Power Query**
* Created calculated attributes and analytical groupings required for HR reporting

---

## Report Structure

**Overview**

* Workforce snapshot and key HR KPIs
* Headcount and attrition insights by organizational structure

**Performance Tracker**

* Employee-level performance history
* Review timelines and satisfaction trends
* Manager and relationship ratings in a single analytical view

**Attrition Analysis**

* Deep-dive into employee turnover drivers
* Attrition breakdowns by organizational, demographic and behavioral factors
